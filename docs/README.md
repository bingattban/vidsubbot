# Arabic Subtitle Telegram Bot

A production-ready Telegram bot that translates subtitles to Arabic using fully offline, open-source tools.

## Features

- 🎬 Support for all websites supported by yt-dlp
- 📝 Automatic subtitle detection and download
- 🤖 AI-powered Arabic subtitle generation
- 🌐 Translation from any language to Arabic
- 📹 Video download in any quality
- 🔒 100% offline processing (no cloud APIs)
- 🕒 Automatic temporary file cleanup (3 hours)
- ⚡ Async processing with progress updates
